# ✅ RESUELTO: Base de Datos SQLite

## Cambio Realizado
- **Antes:** PostgreSQL en Render (expiraba 22 de Abril)
- **Ahora:** SQLite local en el servidor (sin expiración, 100% gratuito)

## Ventajas de SQLite
✅ Completamente gratuito  
✅ Sin expiración  
✅ Funciona perfecto para una app de evento  
✅ Menor consumo de recursos  
✅ Fácil de hacer backup  

## Cómo Funciona
- Base de datos guardada en `db.sqlite3`
- Se guarda automáticamente en el servidor de Render
- No necesita configuración adicional

## Backup de Datos
Si necesitas hacer backup:
```bash
# Descargar la BD
git clone https://github.com/tu-usuario/carreta.git
# El archivo db.sqlite3 estará en la raíz
```
